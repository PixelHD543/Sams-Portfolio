Card Architecture – JavaScript Object Model

The entire game is driven by a central data structure. Each card is a JavaScript object with properties that define its behaviour.

// Example card from Dino Duel TCG data model
{
  name: "T-Rex",
  attack: 8,
  health: 12,
  energyCost: 3,
  ability: {
    name: "Roar",
    effect: "opponent_discard_random",
    description: "Force opponent to discard 1 random card"
  },
  image: "images/trex.png",
  usedAbilityThisTurn: false
}

Ability System – Energy Costs & Effect Dispatch

const abilityEffects = {
  "opponent_discard_random": (target) => {
    let randomIndex = Math.floor(Math.random() * target.hand.length);
    target.hand.splice(randomIndex, 1);
  },
  "heal_self": (player, amount) => {
    player.health = Math.min(player.maxHealth, player.health + amount);
  }
};
