Project Overview

Dino Duel TCG is a fully playable trading card game built from scratch using vanilla HTML, CSS, and JavaScript. Players battle with dinosaur-themed cards, each with unique abilities, energy costs, and attack values.

  • ✅ 42 commits – full Git history showing iterative development
  • ✅ Dynamic card rendering from JavaScript objects
  • ✅ Ability system with energy-cost mechanics
  • ✅ Responsive design – works on desktop and mobile
  • ✅ Deployed via GitHub Pages with custom domain structure

Technical Implementation

Game Architecture:

  • Card data model: JavaScript array of objects (name, attack, health, ability, energy cost, image path)
  • Game state management: Turn-based logic with player health tracking
  • Event handling: Click events for card selection, ability activation, and attacks
  • Visual feedback: CSS transitions for card hover and battle effects

Key Feature – Ability System: Cards with energy costs display a "⚡ ABILITY" button. The system checks available energy before activating special effects (e.g., healing, bonus damage, opponent debuffs).

Challenges & Solutions

Challenge 1: Card effect state management – Tracking which cards have used abilities in a turn. Solution: Added a `usedAbilityThisTurn` flag per card, reset at end of turn.

Challenge 2: Image loading fallbacks – Missing card images broke layout. Solution: Implemented error handling with placeholder images and console logging for debugging.

Challenge 3: Balancing ability costs – Initially energy costs were too high, making abilities unusable. Solution: Iterative playtesting and tweaking values across 10+ commits.
